# Basics Mobile Template

## 项目介绍

Basics Mobile Template是一个基于Vue3、Mockjs和TypeScripe的移动端模版。

## 技术栈

- Vue3
- TypeScript
- Vue Router
- Pinia
- Vant Ui
- Mockjs
- Axios
- SCSS

## 安装

### 克隆项目

```bash
git clone https://github.com/LeoSunQi/basics-mobile-template.git
```

### 进入项目目录

```bash
cd basics-mobile-template
```

### 安装依赖

```bash
npm install
```

### 启动项目

```bash
npm run dev
```
